Η εργασία αποτελείται από δύο directories, με το καθένα να έχει τα δικά του αρχεία και makefiles.
Τα makefiles έχουν εντολή make default, όπου το κομμάτι του client ζητάει ένα directory server που θα πρέπει να βρίσκεται
μέσα στο ίδιο directory με το remoteserver.cpp. Εναλλακτικά μπορείτε να ζητήσετε οποιοδήποτε υπάρχον directory. 
Ο server συνδέεται με τους clients και τους δίνει το blocksize για να μπορέσουν να κάνουν τον σωστό αριθμό επαναλήψεων
όταν διαβάσουν τα blocks. Το κάθε socket μετα αντιστοιχείται με ένα mutex μέσω map για να υπάρχει προσβαση στο σωστό
mutex κάθε φορά που πρέπει να στείλουμε μήνυμα. Υπάρχει επίσης ένα mutex για το Queue και ένα για το out stream.
Το communication thread που αντιστοιχεί στη κάθε σύνδεση βρίσκει τα αρχεία, και τα βάζει σε ένα vector, για να μπορέσει
να στείλει μετά το πλήθος τους στον client.
Ο client κόβει τα directories πριν το τελευταίο, για να δημιουργήσει μόνο αυτό που θα έχει ανοίξει, μετά κόβει το αρχείο
και στέλνει τα υπόλοιπα directories προς δημιουργία, και το αρχείο προς δημιουργεία και γράψιμο.
